# MegaSync Pro
==========

MegaSync Pro is a powerful and user-friendly client for syncing and working with MEGA cloud storage. It gives you full access to premium features without an official subscription, along with higher limits and extra capabilities.

Perfect for heavy MEGA users: people who share large amounts of data, make backups, work with media files, distribute content, or simply want to get rid of free account restrictions.

## Key Features

- Full two-way folder synchronization (just like the official client)
- Support for very large files and huge total volumes
- Significantly increased upload/download speed limits (depending on the build)
- Multiple simultaneous file uploads/downloads
- End-to-end encryption support (same as original)
- Runs in the background + auto-start with Windows/macOS
- Notifications for new files and completed transfers
- Built-in file manager and preview
- Selective sync support (choose which folders to sync)
- Mount MEGA as a network drive (in some builds)
- Optional remote control via web interface
- No ads and no telemetry (in modified versions)

## Quick Start
-----------

1. Download the latest release archive.  
2. Extract the archive to a folder of your choice.  
3. Launch MegaSync Pro.
